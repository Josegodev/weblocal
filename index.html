Create a Local Website

Prerequisites

Before starting, ensure you have the following installed:

  • Node.js
  • npm

Step-by-step Guide

  1. Create a new folder for your project:
    ```bash mkdir my-local-website ```
  2. Navigate into the newly created folder:
    ```bash cd my-local-website ```
  3. Initialize a new Node.js project and install required dependencies:
    ```bash npm init -y npm install express --save ```
  4. Create an `index.html` file in the root of your project folder:
    ```bash echo ' My Local Website

    Welcome to my local website!

    ' > index.html ```
  5. Create a new `app.js` file in the root of your project folder:
    ```bash touch app.js ```
  6. Add the following code to `app.js`:
    ```javascript src/app.js const express = require('express'); const path = require('path'); const app = express(); // Serve static files from the root folder (including index.html) app.use(express.static(__dirname)); // Set the home route to serve our index.html file app.get('/', (req, res) => { res.sendFile(path.resolve(__dirname, 'index.html')); }); // Start the server on port 3000 app.listen(3000); ```
  7. Start your local web server:
    ```bash node app.js ```
  8. Open a web browser and go to `http://localhost:3000`
