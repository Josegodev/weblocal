Setup guiado

Construye el servidor paso a paso

Cada paso incluye el comando o codigo necesario para avanzar sin perder contexto.

  1. Paso 01

    Crea la carpeta del proyecto

    Empieza con un directorio limpio para aislar dependencias y archivos.

    bash
    mkdir backend-master
cd backend-master
  2. Paso 02

    Inicializa Node.js

    Genera package.json, el manifiesto que describe tu proyecto.

    bash
    npm init -y
  3. Paso 03

    Instala Express

    Express sera la capa HTTP que conectara tu navegador con los archivos del proyecto.

    bash
    npm install express --save
  4. Paso 04

    Crea la estructura de archivos

    Separa servidor, pagina y estilos como en un proyecto real.

    backend-master/ ├─ app.js ├─ index.html └─ styles/ └─ main.css
  5. Paso 05

    Programa el servidor

    Sirve archivos estaticos y responde la ruta principal.

    app.js
    const express = require('express');
const path = require('path');

const app = express();

app.use(express.static(__dirname));

app.get('/', (req, res) => {
    res.sendFile(path.resolve(__dirname, 'index.html'));
});

app.listen(3000, () => {
    console.log('Server running at http://localhost:3000');
});
  6. Paso 06

    Ejecuta y valida

    Arranca el proceso y abre la URL local en el navegador.

    terminal
    $ node app.js
Server running at http://localhost:3000

Checklist

  • Node instalado
  • Dependencias en package.json
  • Express sirviendo estaticos
  • localhost respondiendo
Tip de backend

Si cambias app.js, reinicia el servidor o instala una herramienta como nodemon.

El usuario debe sentir que esta operando dentro de un entorno profesional de desarrollo, no leyendo una pagina generica.
