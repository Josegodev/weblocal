Crea un sitio web local

Requisitos previos

Antes de comenzar, asegúrate de tener lo siguiente instalado:

  • Node.js
  • npm

Guía paso a paso

  1. Crea una carpeta para tu proyecto:
    ```bash mkdir mi-sitio-web-local ```
  2. Navega hasta la carpeta recién creada:
    ```bash cd mi-sitio-web-local ```
  3. Inicializa un proyecto de Node.js y instala las dependencias necesarias:
    ```bash npm init -y npm install express --save ```
  4. Crea un archivo `index.html` en la carpeta principal de tu proyecto:
    ```bash echo ' Mi sitio web

    Bienvenido a mi sitio web local!

    ' > index.html ```
  5. Crea un archivo `app.js` en la carpeta principal de tu proyecto:
    ```bash touch app.js ```
  6. Agrega el siguiente código a `app.js`:
    ```javascript src/app.js const express = require('express'); const path = require('path'); const app = express(); // Serve static files from the root folder (including index.html) app.use(express.static(__dirname)); // Set the home route to serve our index.html file app.get('/', (req, res) => { res.sendFile(path.resolve(__dirname, 'index.html')); }); // Start the server on port 3000 app.listen(3000); ```
  7. Arranca el servidor web local:
    ```bash node app.js ```
  8. Abre un navegador y vaya a `http://localhost:3000`
