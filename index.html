Bienvenido a WebLocal

Aprende a crear y ejecutar tu propio sitio web local usando Node.js y Express.

Esta guía práctica te llevará paso a paso desde la configuración inicial hasta tener un servidor web funcionando en tu computadora. No necesitas experiencia previa con Node.js — solo seguir las instrucciones.

Qué aprenderás

  • Configurar un proyecto Node.js desde cero
  • Instalar y usar Express como servidor web
  • Servir archivos estáticos con Express
  • Ejecutar un servidor local en tu máquina

Selecciona la pestaña Guía para comenzar.

Guía paso a paso

Requisitos previos

Antes de comenzar, asegúrate de tener lo siguiente instalado:

  • Node.js
  • npm

Pasos

  1. Crea una carpeta para tu proyecto

    mkdir mi-sitio-web-local

  2. Navega hasta la carpeta recién creada

    cd mi-sitio-web-local

  3. Inicializa un proyecto de Node.js y instala las dependencias necesarias

    npm init -y
npm install express --save

  4. Crea un archivo index.html en la carpeta principal de tu proyecto

    echo '
<!DOCTYPE html>
<html lang="es">

<head>
    <title>Mi sitio web</title>
</head>

<body>
    <h1>Bienvenido a mi sitio web local!</h1>
</body>

</html>' > index.html

  5. Crea un archivo app.js en la carpeta principal de tu proyecto

    touch app.js

  6. Agrega el siguiente código a app.js

    const express = require('express');
const path = require('path');

const app = express();

// Serve static files from the root folder (including index.html)
app.use(express.static(__dirname));

// Set the home route to serve our index.html file
app.get('/', (req, res) => {
    res.sendFile(path.resolve(__dirname, 'index.html'));
});

// Start the server on port 3000
app.listen(3000);

  7. Arranca el servidor web local

    node app.js

  8. Abre un navegador y ve a http://localhost:3000

    ¡Deberías ver tu sitio web funcionando!

Acerca de este proyecto

Este sitio es una guía tutorial diseñada para ayudar a principiantes a crear y ejecutar un sitio web local usando tecnologías modernas del lado del servidor.

Tecnologías utilizadas

  • HTML5 Estructura y contenido de la página
  • CSS3 Diseño visual y presentación
  • JavaScript Interactividad y lógica de navegación por pestañas
  • Node.js Entorno de ejecución del lado del servidor
  • Express Framework web para Node.js

Propósito

El objetivo de este tutorial es proporcionar una introducción clara y práctica al desarrollo backend con Node.js y Express, permitiendo a cualquier persona crear su primer servidor web local en pocos minutos.
